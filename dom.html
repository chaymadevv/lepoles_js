D.O.M : Document Object Model

Lorsque le navigateur charge une page HTML, javascript crée un objet pour chaque élément HTML de cette page. C'est ce qu'on appelle le DOM.

Dans cette représentation orienté objet d'une page web, tous les éléments de la page sont des propriétés d'un objet : window Pour simplifier, l'objet window représente l'onglet du navigateur. Comme tous les éléments sont des éléments de cet objet, on n'a pas besoin de le préciser, il est implicite.

L'objet qui représente la page HTML est document
