Structures itératives

Boucle FOR

Pour exécuter un certain nombre de fois une suite d'instructions, on utilise la boucle FOR. On va définir une variable qui va servir à compter le nombre de tour (=itération). On va définir une condition qui va permettre d'arrêter la boucle, quand cette condition sera fausse. On va définir une incrémentation (ou décrementation) de la variable qui sert à compter les itérations.

Syntaxe :

for(initialisation ; condition ; incrémentation) {
... bloc d'instructions ...
}

Boucle WHILE (tant que)

  • Syntaxe :

    while( condition ) {
    ...bloc d'instructions ...
}

  • Explication :

    Tant que la condition est vrai, on continue la boucle. ⚠ aux boucles infinies

Boucle DO WHILE

  • Boucle DO WHILE (faire tant que) 
        SYNTAXE :
        do {
            bloc d'instructions
        } while (condition)

    NB : la différence entre cette boucle et la boucle WHILE, c'est que la condition est testé après le 
            bloc. Donc, quoiqu'il arrive, le bloc d'instructions sera exécuté au moins une fois pour la 
            boulce DO WHILE.


    En résumé, on utilise la boucle FOR lorsqu'on connaît le nombre d'itérations (=tours) à faire.
    Si on ne sait pas combien de tours ou si la condition d'arrêt n'est pas lié au nombre de tour,
    on utilise la boucle WHILE.
    Si on doit exécuter le bloc d'instructions au moins 1 fois, quelque soit la valeur de la condition,
    alors on utilise la boucle DO WHILE.

    NB : les 3 boucles sont interchangeables si on définit bien la condition de sortie de la boucle.

FOREACH

La boucle FOREACH est une boucle utilisée pour les tableaux dans différents langages de programmation. En JS, cette boucle est liée aux objets de type array.
Syntaxe : 

        tableau.forEach( function(valeur, indice) {
            ... instructions ...
        })

Dans FOREACH, on va déclarer une fonction qui va s'appliquer sur toutes les valeurs du tableau.

FOR OF

Une autre version de la boucle FOREACH en JS est la boucle FOR OF

Syntaxe: 

            for(const valeur of tableau) {
                ... instructions ...
            }

Pour chaque valeur du tableau, on va exécuter les instructions de la boucle.

Ici on ne récupère pas les indices, uniquement les valeurs qui sont affectées à une nouvelle variable (dans l'exemple, c'est la variable jour).
